Hilfe

sysHUBFindr richtet sich vorrangig an Entwickler eines sysHUB Servers. Viele der in diesem Tool verwendeten Begriffe setzen ein grundlegendes Verständnis des Servers voraus. Diese Hilfeseite versucht die wichtigsten Funktionen zu erklären.

Allgemeine Benutztungshinweise

Im oberen Bereich der Webseite befindet sich die Navigationsleiste. Dort finden sich auf der linke Seite Verknüpfungen zur Startseite und zu einer Statistik-Seite. Von der Startseite aus kann auch die Suche gestartet werden.


Auf der rechten Seite der Navigationsleiste befinden sich zum einen die Suchleiste mit einem Optionen-Dropdown und dem Button zum Starten der Suche. Das Optionen-Dropdown ermöglicht die Angabe, in welchen Bereichen eines sysHUB-Servers überhaupt gesucht werden soll.
Ganz rechts befindet sich ein sogenanntes Burger-Menü, das Symbol mit den drei Streifen. Dieses Menü öffnet weitere Optionen, unter anderem um zu dieser Hilfeseite zu gelangen.

  • Cache leeren: Findr speichert viele Daten aus dem sysHUB im Browser zwischen. Damit müssen viel weniger Daten nach einer erfolgten Suche neu heruntergeladen werden und das Ergebniss kann schneller angezeigt werden.
  • Deutsch/Englisch: Ändert die Anzeigesprache des Findr. Sämtliche Elemente aus dem sysHUB (z.B. Workflownamen) werden nicht übersetzt.
  • Hilfe: Öffnet diese Hilfe.
  • Über: Öffnet eine Seite mit weiteren Informationen, was Findr überhaupt ist und wo die Weiterentwicklung stattfindet.
  • Findr auf Github: Öffnet einen neuen Browser-Tab zur Code-Plattform Github. Je nach Konfiguration des Findr steht diese Option nicht zur Verfügung.
  • sysHUB web client: Startet den sysHUB web client in einem neuen Browser-Tab. Je nach Konfiguration des Findr steht diese Option nicht zur Verfügung.
  • Abmelden: Meldet Dich vom Server ab und beendet die Sitzung im Findr, du wirst dann auf die Anmeldeseite zurückgeschickt. Je nach Konfiguration des Findr steht diese Option nicht zur Verfügung.

Help

sysHUB Findr is primarily targeted at developers of a sysHUB server. Many terms used in this tool presuppose a basic understanding of the server. This help page aims to explain the key features.

General Usage Tips

At the top of the webpage is the navigation bar. On the left side, you'll find links to the homepage and a statistics page. The search function can also be initiated from the homepage.


On the right side of the navigation bar are the search bar with an options dropdown and the button to start the search. The options dropdown allows you to specify which areas of a sysHUB server to search in.
On the far right is a so-called burger menu, the icon with three stripes. This menu opens additional options, including access to this help page.

  • Clear cache: Findr stores many sysHUB data in the browser. This reduces the amount of data that needs to be re-downloaded after a search, allowing faster display of results.
  • German/English: Changes the display language of Findr. All sysHUB elements (e.g., workflow names) remain untranslated.
  • Help: Opens this help page.
  • About: Opens a page with more information about what Findr is and where its development takes place.
  • Findr on Github: Opens a new browser tab to the code platform Github. This option may not be available depending on the configuration of Findr.
  • sysHUB web client: Launches the sysHUB web client in a new browser tab. This option may not be available depending on the configuration of Findr.
  • Logout: Logs you out of the server and ends your session in Findr, redirecting you to the login page. This option may not be available depending on the configuration of Findr.
