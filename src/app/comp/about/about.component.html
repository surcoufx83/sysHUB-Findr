Über sysHUB Findr

sysHUBFindr ist ein Projekt das im Zuge der Arbeit mit der Software NT-ware uniFLOW sysHUB (nachfolgend kurz sysHUB, siehe auch https://www.syshub.global/) entstand. sysHUB ist eine Java-basierte flexible Workflow- und Output-Management-Anwendung die Prozesse automatisieren und überwachen kann. Durch die integrierten low-code Workflows werden Anforderungen schnell erreicht.

Aber kein System macht alles richtig, so gibt es auch Tücken im sysHUB. Findr ist das Ziel, eine der Lücken zu schließen: die unzureichende Suche und Nachverfolgbarkeit innerhalb der Entwicklungsumgebung. Findr wird open-source entwickelt. Ideen und Fehler können als Issues eingebracht werden, ebenso eigene Code-Vorschläge.

  1. Einfach zugängliche Suchfunktion für sämtliche Bereiche innerhalb eines sysHUB Systems
  2. Caching Mechanismus zur schnellen Anzeige der Ergebnisliste
  3. Grafisch dargestellte Workflows
  4. Verfolgbarkeit von Variablen

About sysHUB Findr

sysHUB Findr is a project that emerged in the course of working with the software NT-ware uniFLOW sysHUB (hereinafter referred to briefly as sysHUB, see also https://www.syshub.global/). sysHUB is a Java-based flexible workflow and output management application that can automate and monitor processes. Its integrated low-code workflows quickly meet requirements.

However, no system is perfect, and sysHUB has its challenges. Findr aims to bridge one of these gaps: the insufficient search and traceability within the development environment. Findr is being developed as open-source. Ideas and bugs can be submitted as issues, as well as one's own code suggestions.

Currently, sysHUB Findr offers the following features:

  1. Easy-to-access search function for all areas within a sysHUB system
  2. Caching mechanism for quick display of search results
  3. Graphically represented workflows
  4. Traceability of variables
